# Linked-List
Pair programming demo purpose

## Things to look into:
Single linked list
* is empty
* add value
* get last value
* get first value
* get value (Int pos)
* add head value
* clear
* size
* iterator
    * next
    * hasNext
* constructor - list as parameter
* insert(Int pos)
* remove(Int pos)

## Bonus
* detect loop
* double link list - add back

##Requires:
1. Java (1.8 but you can switch to 1.6)
1. gradle (wrapper is provided)
